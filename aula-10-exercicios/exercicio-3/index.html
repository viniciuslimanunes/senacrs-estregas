Template JavaScript

Utilize o arquivo "index.js" para programar.

Abra o arquivo "index.html" no navegador para testar os códigos.

Clique com o botão direito do mouse em qualquer ponto da tela e escolha a opção "Inspecionar" para abrir a área para desenvolvedores do navegador.

Para acessar detalhes do código JavaScript, acesse a aba "Console" do Inspetor.

Recarregue a página sempre que precisar reiniciar o código.
